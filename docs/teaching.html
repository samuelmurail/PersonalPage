Jupyter Training Course

Jupyter notebooks allow scientists to interact with computer tools, code and data via a web browser. Jupyter notebooks can be used with any programming language and can also be integrated with the command line tools.
This course aims to introduce participants to Jupyter and related technologies. This can help improve the productivity of daily research, teach a class of students, and adapt the research to the principles of FAIR data (findable, accessible, interoperable, reusable).
The RPBS platform has offered a two-day course on Jupyter on June 5th and 6th 2019. The course is aimed at bio-informaticians with prior expertise in data analysis, who are interested to learn about Jupyter notebooks.
